# setup-project-worklog

Initialize a reusable resumable worklog baseline for software repositories.

## What this skill does

This skill sets up a lightweight project operating system with:

- `AGENTS.md` guidance for worklog, planning, and execution
- `notes/worklog/INDEX.md` for active, blocked, paused, and recently completed work
- `notes/worklog/LESSONS.md` for reusable repo-wide lessons
- `notes/worklog/templates/FEATURE.md` for feature-level tracking
- `notes/plans/` and `notes/worklog/archive/` for longer plans and completed work

## Files

- `SKILL.md` — skill definition and workflow guidance
- `reference.md` — starter templates and baseline file bodies

## Typical use cases

- Adding resumable task tracking to an existing repo
- Standardizing agent instructions and work-in-progress notes
- Creating durable handoff context across sessions

## Status

Initial version. Intended for lightweight repo-wide worklog setup.
