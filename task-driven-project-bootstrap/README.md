# task-driven-project-bootstrap

Create a reusable task-driven project operating system for software repos.

## What this skill does

This skill scaffolds a practical workflow for AI-assisted development with:

- `AGENTS.md` (team/agent operating instructions)
- `PROGRESS.md` (current status snapshot and handoff context)
- `tasks.md` (task list and execution queue)
- helper bootstrap script and templates

## Files

- `SKILL.md` — skill definition and usage guidance
- `scripts/bootstrap_task_driven_project.sh` — setup script
- `templates/AGENTS.md.tpl`
- `templates/PROGRESS.md.tpl`
- `templates/tasks.md.tpl`

## Typical use cases

- Starting a new repo with strong execution hygiene
- Standardizing workflow across multiple projects
- Improving session continuity and handoff quality
